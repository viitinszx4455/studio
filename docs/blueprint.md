# **App Name**: Nexora Portfolio

## Core Features:

- Home Section: Replicates the hero section of the Nexora website, including the gradient background and client logos.
- Services Section: Mimics the services section, displaying the services offered with a similar design and layout.
- Projects Section: Duplicates the projects section to showcase previous work with project links to the original sites like FortInvest(https://fortinvest.netlify.app/) and Liubarberia(https://liubarbeariaa.netlify.app/).
- Workflow Section: Recreates the workflow section, visually representing the steps of the design and development process.
- FAQ Section: Copies the frequently asked questions section, providing information on common inquiries about the agency's services.
- Contact Section: Adds a contact form that allow the users to contact the Nexora agency, by redirecting to the provided whastapp link: https://wa.me/5562991210362?text=Ol√°! Quero criar meu site personalizado.

## Style Guidelines:

- Primary color: Deep purple (#531CB3) for sophistication and trust.
- Background color: Dark grayish blue (#13161F) to provide contrast and highlight content.
- Accent color: Soft lavender (#A7BDFF) as a secondary highlight for interactive elements.
- Headline font: 'DM Serif Display' for titles and important text; a serif for elegance.
- Body font: 'Manrope' for body text and paragraphs; a sans-serif for readability.
- Lucide icons: selected from the Lucide icon set for a consistent and modern feel, for example rocket, megaphone and wrench.
- Consistent container width: Maintain a consistent container width across all sections of the page.
- Gradient animation: Create an subtle 'ultraSlowBlueGradient' to animate the buttons for added visual interest.
